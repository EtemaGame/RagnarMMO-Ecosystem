# RagnarMMO Docs

Documentacion viva del ecosistema modular.

## Documentos activos

- `ARCHITECTURE.md`: reglas actuales de arquitectura, ownership por modulo y dependencias opcionales.
- `PORT_STATUS.md`: estado real del port desde `original-source`, que falta y que comandos usar para validar.
- `LEGACY_COMPATIBILITY.md`: aliases e IDs legacy que no deben eliminarse sin migracion.
- `STABILIZATION.md`: guardrails de estabilizacion y smoke checklist.

## Documentos retirados

Se consolidaron `ECOSYSTEM.md`, `MODULE_EXTRACTION_MAP.md`, `MODULAR_PORT_CHECKLIST.md` y `PORT_AUDIT.md` porque repetian informacion y quedaban obsoletos entre si. La arquitectura canonica vive ahora en `ARCHITECTURE.md`; el avance del port vive en `PORT_STATUS.md`.
